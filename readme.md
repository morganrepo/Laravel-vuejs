# Single Page Application with Laravel 5.3 and Vue.js 2.1.x

### Frameworks and Libraries:

- Laravel 5.3
- Vue.js 2.1.x
- Vue Router

### Provides:

- CRUD
- Single Page Application
- Pagination
- Sorting by columns
- Filter relations



### Installation
`git clone https://github.com/codekerala/spa-laravel-vuejs.git`

`cd spa-laravel-vuejs`

`composer install`
